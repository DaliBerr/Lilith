# Localization Data

`StringTables` contains runtime string table packs loaded by the `Localization` Addressables label.

`JsonPatches` contains external JSON patches loaded by the `LocalizationJson` Addressables label. Patch files use the existing `domain` + `patches[id]` shape consumed by `Vocalith.Localization.LocalizedJsonUtility`.

For non-technical localization work, use the collaborator guide:

`Docs/LocalizationGuide.md`
